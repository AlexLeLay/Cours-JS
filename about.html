1. Introduction à JavaScript

C'est quoi le JavaScript ?

Le JavaScript (JS) est un langage de programmation qui donne de l'interactivité aux pages web.

  • HTML = structure ou le squelette
  • Le CSS = style (les couleurs, la mise en page, etc)
  • JS = logique (les réactions à un utilisateur, des calculs, et la dynamique de la page)

Où s'execute le JS ?

  • Dans le navigateur, (Chrome, FireFox, Safari, etc)
  • Sur un serveur qui tourne avec Node.js -> il ne sera plus un langage front-end mais back-end, comme le PHP ou Python

À retenir : même language mais deux environnements différents

Premier pas : console, balise script

  1. Ouvrir votre navigateur préféré
  2. Click droit -> inspecter, ou si vous êtes sur Windows F12
  3. Aller dans l'onglet console
