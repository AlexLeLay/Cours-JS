SetTimeout() et SetInterval()

SetTimeout()

ça execute une fonction une seule fois après un certain temps

            
                setTimeout(function() {
                    console.log("3 secondes se sont écoulées !");
                }, 3000)

SetInterval()

Execute une fonction de manière répétée à intervalle régulier

            
                setTimeout(function() {
                    console.log("3 secondes se sont écoulées !");
                }, 3000)

clearTimeout() et clearInterval()

clearTimeout()

Pour pouvoir supprimer vos timers, il faudra les mettre dans une variable

            
                const timer = setTimeout(function() {
                    console.log("3 secondes se sont écoulées !");
                }, 3000)

                clearTimeout(timer); // C'est plus intéressant dans un évènement

                const interval = setInterval(function() {
                    console.log("1 seconde vient de passer");
                }, 1000)

                clearInterval(interval)
